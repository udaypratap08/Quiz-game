# Quiz-game

A simple and interactive Quiz Game built using **HTML, CSS, and JavaScript**. The game presents multiple-choice questions, checks user answers instantly, keeps track of the score, and displays the final result at the end.

---

## 🚀 Features

- 📝 Multiple-choice questions
- ✅ Instant answer validation
- 📊 Real-time score tracking
- 🔄 Next question navigation
- 🏆 Final score display
- 📱 Responsive design
- 🎨 Clean and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## 🎮 How to Play

1. Click **Start Quiz**.
2. Read each question carefully.
3. Select the correct option.
4. Click **Next** to continue.
5. After completing all questions, your final score will be displayed.
